# 💸 Money Mind Dashboard

A modern fintech dashboard that helps users track expenses, manage budgets, and understand their financial habits through clean and interactive visualizations.

---

## 🚀 Features

- 📊 Interactive dashboard with financial overview  
- 📈 Charts for tracking expenses and categories  
- 🧾 Add and manage expenses easily  
- 🎯 Savings goals with progress tracking  
- 💡 Smart insights based on spending behavior  
- 📱 Fully responsive design  

---

## 🛠️ Tech Stack

- React / Next.js  
- Chart.js  
- Firebase (for future integration)
